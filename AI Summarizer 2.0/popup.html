Set Hugging Face API Key

To use the Hugging Face model, you need an API key. Follow these steps to obtain your API key:

  1. Visit the Hugging Face API Tokens Page.
  2. Log in or create an account if you don’t have one.
  3. Generate a new token if you don’t already have one.
  4. Copy the token and paste it below.
Your API key will be stored securely and used to access Hugging Face models.
